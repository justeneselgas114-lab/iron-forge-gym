
# IRON FORGE - Premium Fitness Club

A high-performance fitness gym website with a modern aesthetic and AI-powered training assistant.

## Tech Stack
- **React 19**
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **Google Gemini API** (ForgeAI Assistant)

## Deployment to Vercel
1. Push this code to a GitHub repository.
2. Connect the repository to Vercel.
3. Add your `API_KEY` in the Vercel Dashboard -> Settings -> Environment Variables.
